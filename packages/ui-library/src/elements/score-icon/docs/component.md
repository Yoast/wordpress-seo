The ScoreIcon component is a visual indicator that displays score-based feedback to users. It represents different quality or performance levels through either emoji icons or simple colored circles, making it easy for users to quickly understand score assessments at a glance.

## Usage scenarios

ScoreIcon is commonly used in:
- **SEO analysis:** Displaying SEO scores for content.
- **Readability analysis:** Indicating the readability level of text content.
- **Inclusive language analysis:** Highlighting the use of inclusive language in content.
- **Feedback systems:** Providing instant visual feedback on user actions or content quality.

## Score types

The component supports four score levels:
- **good:** Indicates excellent or passing status (green color with a happy emoji).
- **ok:** Represents acceptable or needs improvement status (orange color with a neutral emoji).
- **bad:** Shows poor or failing status (red color with a sad emoji).
- **na:** Indicates that no score is available or applicable (gray color with no emoji).

## Display modes

The ScoreIcon can be displayed in two modes:
- **Emoji mode (default):** Shows expressive emoji icons (happy, neutral, or sad faces) that provide clear emotional context.
- **Circle mode:** Displays simple solid colored circles, offering a more minimal and subtle visual indicator.

If an invalid or no score is provided, the component displays a gray circle as a fallback indicator.
